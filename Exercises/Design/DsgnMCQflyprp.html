Which of these is not a defining property of a flyweight?

"It is small"
  • "It is stateless"
  • "It is a singleton"
  • "It is reused"

There is only one copy of the flyweight (that is shared).

Since a flyweight is reused, the implementor cannot count on its ability to store state (since all occurrances are the same.

It doesn't matter how big the flyweight is.
