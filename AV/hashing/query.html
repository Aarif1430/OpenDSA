This shows how to access a specific hashing function with a collision resolution using queries (as named by JSAV). Here is the list of numbering to use. The number corresponds to the number to pass in the URL.

  1. Simple Mod Hash
  2. Binning Hash
  3. Mid Square Hash
  4. Simple Hash for Strings
  5. Improved Hash for Strings
  1. Linear Probing
  2. Linear Probing by Steps
  3. Pseudo-random Probing
  4. Quadratic Probing
To use it, the frame's URL should be as follows: hash.html?method=NUM&collision=NUM Use hash.html to be able to use them all. For example, here is a visualization where you can use choose your own options:
